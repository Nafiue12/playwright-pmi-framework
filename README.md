# Playwright PMI Framework

Professional UI Automation Framework using Playwright.

## Features

- UI Automation
- API Testing
- Page Object Model (POM)
- HTML Reporting
- Allure Reporting
- GitHub Actions CI/CD
- Screenshot Capture
- Cross Browser Testing

## Technologies

- Playwright
- JavaScript
- Node.js
- Allure Report
- GitHub Actions

## Installation

```bash
npm install
npx playwright install


Run Tests
npx playwright test
Open HTML Report
npx playwright show-report

Generate Allure Report
npx allure generate ./allure-results --clean
npx allure open
